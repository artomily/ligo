## Typography — Manrope + JetBrains Mono

- **Display + body:** Manrope
- **Mono (numbers, addresses, USDT amounts):** JetBrains Mono

Wired via `next/font/google` in `app/layout.tsx`. CSS variables: `--font-sans`, `--font-mono`.

## Gradients

Two gradients derived from the palette, deliberately restrained (premium mood). Exposed as Tailwind utilities `bg-brand-bg` and `bg-brand-accent`.

### Background gradient (subtle)

```css
--gradient-bg: linear-gradient(135deg, oklch(0.98 0.005 255) 0%, oklch(0.955 0.012 270) 100%); /* light */
--gradient-bg: linear-gradient(135deg, oklch(0.14 0.015 255) 0%, oklch(0.17 0.02 270) 100%);   /* dark */
```

**Use for:** hero sections, page headers, large empty areas.
**Do not use for:** body text backgrounds, form fields, cards.

### Accent gradient

```css
--gradient-accent: linear-gradient(to right, oklch(0.5 0.14 255) 0%, oklch(0.52 0.14 285) 100%);   /* light */
--gradient-accent: linear-gradient(to right, oklch(0.72 0.14 255) 0%, oklch(0.74 0.14 285) 100%);  /* dark */
```

**Use for:** the landing hero CTA, feature-card headers, premium badges. At most one gradient per screen.
**Do not use for:** body text backgrounds, checkout/payment buttons (money UI stays solid).

## Tone and voice

### Words to use

Confident, understated, considered — with human warmth. Short sentences and whitespace do the talking. Second person ("you") and contractions are fine; Ligo is a community product, not a bank. Describe outcomes, not effort: "Your jersey ships from Jakarta" beats "seamlessly connect with sellers worldwide". Numbers are part of the voice — show amounts, member counts, and delivery times plainly in mono.

### Words to avoid

Crypto jargon in user-facing copy: "on-chain", "web3", "wallet-native", "gas", "DeFi". USDT is mentioned as a currency, not a technology. Also banned: "luxury", "exclusive", "unlock", "seamless", "powerful", "revolutionary", hype exclamation marks, and stacked adjectives.

### Voice example

> "45 USDT. Ships from Jakarta in 3 days. Sold by a fellow Indonesia Fans member."

## Usage dos and don'ts

**Do:**

- Use the shadcn CSS variables (`bg-background`, `text-foreground`, `bg-primary`) everywhere. Never hardcode hex.
- Keep the 4 px spacing grid (`gap-2`, `p-4`, `px-6`).
- Use `font-mono tabular-nums` for any number that updates in place (USDT balances, prices, member counts).
- Test every component in both light and dark mode before considering it done.

**Don't:**

- Hardcode colors (`#FF0000`, `rgb(...)`) in component files.
- Use `transition: all` — always specify `transition-colors`, `transition-transform`, etc.
- Mix more than one gradient on the same screen.
- Use neon, glows, or "hacker" aesthetics — Ligo references Discord/Airbnb/Stripe/Linear/Notion, never a DeFi terminal.
- Override shadcn tokens per-component — if a color is wrong, fix it in `globals.css` once.
